# zdppy_mysql
使用python操作MySQL,同时支持同步版本和异步版本,支持事务

项目地址: https://github.com/zhangdapeng520/zdppy_mysql

## 功能清单
同步版本的MySQL
- execute() 执行SQL语句
- executes() 批量执行SQL语句
- fetchone() 查询单条数据
- fetchall() 查询所有数据

异步版本的MySQL
- execute() 执行SQL语句
- executemany() 执行批量插入的SQL语句
- executes() 批量执行SQL语句
- fetchone() 查询单条数据
- fetchall() 查询所有数据
